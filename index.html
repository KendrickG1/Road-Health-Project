Key Features

📸 Image Capture

Users take photos directly from their smartphone camera or upload existing images of road hazards.

📍 Auto-Location

GPS automatically captures the exact location of the hazard with reverse geocoding for address details.

🤖 AI Classification

Teachable Machine model automatically identifies hazard type with confidence scoring.

⚠️ Severity Detection

Intelligent algorithm determines severity level (Low/Medium/High) based on hazard type and AI confidence.

🗺️ Interactive Map

Google Maps integration with color-coded pins showing all hazards and their current status.

🔄 Status Tracking

Road crews can update hazard status from Reported → In Progress → Fixed with timestamps.

📊 Analytics Dashboard

Comprehensive statistics and charts showing trends, response times, and completion rates.

👷 Road Crew Portal

Secure login for maintenance teams to manage and update hazard statuses efficiently.

User Workflow

👤 Community Member (Reporting)

  1. Discover Hazard: Find a pothole, crack, debris, or damaged sign
  2. Open App: Launch the web app on smartphone or computer
  3. Capture Photo: Take a clear picture of the hazard
  4. Auto-Location: GPS automatically captures coordinates (or enter manually)
  5. Fill Details: Enter road name and select hazard type
  6. Submit: AI processes image and determines severity
  7. Confirmation: Receive confirmation and view on map

👷 Road Crew (Managing)

  1. Login: Access road crew dashboard with credentials
  2. View Reports: See all active hazards on map and list view
  3. Filter & Sort: Filter by severity, type, or location
  4. Review Details: Click hazard to see full information and photos
  5. Update Status: Mark as "In Progress" when starting work
  6. Mark Fixed: Update to "Fixed" when repair is complete
  7. View Analytics: Check dashboard for statistics and trends

Data Flow Summary

📥 Input: Photo + GPS + Road Name + Type Selection

🔄 Processing: Upload → AI Classification → Severity Detection → Database Storage

📤 Output: Map Pin + Database Entry + Dashboard Display

🔁 Updates: Road Crew Login → Status Change → Database Update → Map Refresh

📊 Analytics: Aggregate Data → Generate Statistics → Display Insights

Color Coding System

🟢 Green - Low Severity

Minor issues that don't pose immediate danger (small cracks, minor debris)

🟡 Yellow - Medium Severity

Noticeable hazards that need attention (moderate potholes, scattered debris)

🔴 Red - High Severity

Dangerous conditions requiring immediate action (large potholes, fallen signs, major debris)

Status States

🔴 Reported

Newly submitted hazard awaiting review by road crew

🟡 In Progress

Road crew has acknowledged and is working on the repair

✅ Fixed

Hazard has been repaired and is no longer a concern

📦 Archived

Fixed hazards moved to historical records for analytics
